# Nutricion Asistente IA

Aplicativo web de capacitación para el Área de Calidad del Servicio de Nutrición del Hospital HNERM.

## Uso

1. Abrir el archivo `/home/runner/work/nutricion-asistente-ia/nutricion-asistente-ia/index.html` en un navegador.
2. Registrar asistencia con nombre completo, tipo, área y celular (PIN único).
3. Completar módulos, rendir examen y descargar certificado PDF si se aprueba.
4. Ingresar al panel administrador con PIN `1206` para generar PIN de reingreso y descargar reporte CSV de asistencia y notas.
